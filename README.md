# sdk-template

EventBus + doT 模板引擎 + fetch

## 修改配置项目

config/constant.js 中进行修改 SDK 的


命名空间：
```
NAME_SPACE
```

调用方式：

```
SDK_EXE
```

调用选项：

```
SDKOptions
```

## 安装依赖

```
npm install
```

### 本地项目启动

```
npm run dev
```

### 项目构建

```
npm run build
```

在 build 文件夹中生成 sdk.min.js

### 用于测试环境

```
npm run test
```
